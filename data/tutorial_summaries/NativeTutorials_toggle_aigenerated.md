# Model Explanation: Toggle

## One-sentence summary

Toggle demonstrates toggle switch.

## What the model shows

Toggle shows toggle switch by focusing on synthesis of x according to a user defined rate law. The tutorial connects that event to readouts that make the modeled behavior visible.

## Biological story

Toggle is a tutorial model centered on toggle switch. Its narrative moves from synthesis of x according to a user defined rate law to 0 is a source, so the model teaches a specific mechanism rather than a broad pathway survey.

## Main biological players

X, Y

## Mechanism in plain English

Toggle uses X, Y as the main actors. First, synthesis of x according to a user defined rate law. Next, 0 is a source. Finally, clearance of x, which is what links the mechanism to the plotted readouts.

## Key modeled events

- Synthesis of X according to a user defined rate law.
- 0 is a source.
- Clearance of X.

## What the model measures

The readouts track X tot, Y tot. These measurements are useful for seeing whether synthesis of x according to a user defined rate law changes the amount of free, bound, active, inactive, product-like, or complexed material.

## Expected behavior in plots

For Toggle, compare readouts connected to synthesis of x according to a user defined rate law with readouts connected to 0 is a source. Separation between those curves indicates that the tutorial mechanism is changing the system in stages rather than all at once.

## Caveats

Toggle is scoped to toggle switch. It should be read as a focused tutorial or validation example, not as a comprehensive biological pathway model.
