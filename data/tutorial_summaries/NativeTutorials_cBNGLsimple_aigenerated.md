## What the model shows

cmodel simple shows a simplified signal transduction model including the following processes. It is useful because it keeps attention on receptor ligand binding while still connecting that step to the model readouts.

## Biological story

cmodel simple is a tutorial model focused on a simplified signal transduction model including the following processes. The biological narrative is built around receptor ligand binding, with phosphorylated receptor internalization providing the next layer of behavior rather than a generic pathway-wide description.

## Mechanism in plain English

cmodel simple begins with receptor ligand binding. It then adds phosphorylated receptor internalization, which changes how L, R, TF, DNA, mRNA, P contribute to the tutorial behavior. The third modeled step, receptor mediated transcription factor phosphorylation, connects the upstream event to the readouts a user would inspect after simulation.

## Key modeled events

- Receptor ligand binding.
- Phosphorylated receptor internalization.
- Receptor mediated transcription factor phosphorylation.

## What the model measures

The readouts include L Bound PM, L Bound EM, Rp tot, Rp PM, Rp EM, TF nuc, and additional related outputs. They show how cmodel simple distributes molecules among active, inactive, bound, free, or product-like states.

## Expected behavior in plots

For cmodel simple, inspect readouts that report receptor ligand binding and compare them with readouts affected by phosphorylated receptor internalization. A visible delay, plateau, or separation between those outputs indicates how receptor mediated transcription factor phosphorylation changes the tutorial dynamics.

## Caveats
